# PDF 文档优化专家 (pdf-optimizer)

## 📖 简介

这是一个专为 Aone Copilot 设计的 Skill，旨在帮助用户全方位优化 PDF 文档。无论是印刷报告、屏幕阅读、长期归档还是无障碍合规，该 Skill 都能从**结构完整性、内容可读性、文件性能、合规无障碍**四个维度提供专业的诊断与优化建议，显著提升信息获取效率与专业度。

## ✨ 核心功能

-   **书签/大纲修复**：校正层级错乱、缺失或重复的导航条目
-   **OCR 文本层补全**：为扫描件添加可搜索/可复制的文本层
-   **字体嵌入验证**：确保跨设备显示一致，杜绝乱码
-   **智能压缩**：根据用途（印刷/屏幕/归档）选择最优压缩策略，体积减少 30%-70%
-   **Fast Web View**：启用线性化，首屏加载速度提升 80%+
-   **PDF/A & WCAG 合规**：验证长期归档标准与无障碍访问要求
-   **分级诊断报告**：按 🔴高/🟡中/🟢低 优先级输出问题清单、工具命令与量化对比

## 🚀 快速开始

### 1. 安装 Skill
将本目录复制到你的项目或全局 Skill 目录中：
```bash
# 项目级（仅当前项目可用）
cp -r pdf-optimizer /your-project/.aone_copilot/skills/

# 全局级（所有项目可用）
cp -r pdf-optimizer ~/.aone_copilot/skills/
```

### 2. 使用方式
在 Aone Copilot 对话中直接输入以下指令即可触发：

-   "帮我优化这份 PDF"
-   "PDF 太大了怎么压缩"
-   "PDF 无法搜索文字"
-   "检查 PDF 是否符合 PDF/A 标准"
-   直接上传 `.pdf` 文件并附带优化需求

### 3. 文件支持说明
-   **支持格式**：`.pdf`（所有版本 1.3-2.0）
-   **文件大小**：建议 ≤200MB；超过 500MB 的文件可能因解析超时失败，建议先拆分或预处理
-   **加密文件**：支持密码保护的 PDF，但需提供密码才能进行深度优化；DRM 保护文件无法处理
-   **扫描件**：可识别纯图片 PDF，推荐 OCR 工具及参数
-   **表单 PDF**：支持交互式表单字段检查与无障碍优化

### 4. 输出示例
Skill 将返回结构化的诊断报告，包含：
-   文档用途、当前大小、合规目标
-   整体评分（0-100）
-   分级问题清单（含页码定位）
-   具体工具推荐与命令行操作
-   优化前后的量化对比表

## 📋 适用场景

| 场景 | 优化重点 | 典型问题 |
|------|----------|----------|
| 印刷出版 | CMYK 色彩、300dpi、字体嵌入、PDF/X | RGB 图片、低分辨率、字体缺失 |
| 屏幕分发 | sRGB、150dpi、Fast Web View、压缩 | 文件过大、加载慢、颜色偏差 |
| 长期归档 | PDF/A-2b/3u、元数据、无外部引用 | 透明效果、JavaScript、未嵌入字体 |
| 无障碍合规 | Tagged PDF、Alt Text、阅读顺序、对比度 | 标签缺失、图片无替代文本、Tab 顺序错乱 |
| 扫描文档数字化 | OCR、文本层、书签重建、去噪 | 无法搜索、倾斜、背景污渍 |

## ⚙️ 自定义配置

如需调整优化侧重点，可编辑 `SKILL.md` 中的以下部分：

-   **角色定位**：修改为特定领域专家（如"学术出版 PDF 顾问"、"政府公文合规审查员"）
-   **核心能力**：增删特定维度的检查项（如增加"数字签名验证"、删除"表单可访问性"）
-   **输出格式**：调整报告模板以匹配机构合规检查清单
-   **触发条件**：添加更多激活关键词或合规标准名称

## 🛠️ 推荐工具链

本 Skill 推荐的工具均为开源或行业标准，非商业绑定：

-   **Adobe Acrobat Pro**：业界标杆，适合复杂交互与预检（付费）
-   **Ghostscript**：命令行压缩/转换/字体嵌入神器（开源，GPL）
-   **qpdf**：线性化、解密、结构修复首选（开源，Apache-2.0）
-   **ocrmypdf**：基于 Tesseract 的 OCR 文本层添加工具（开源，MPL-2.0）
-   **pdftk / pdftools**：书签操作、合并拆分、元数据编辑（开源）
-   **VeraPDF**：PDF/A 合规验证官方工具（开源，EUPL）
-   **PAC (PDF Accessibility Checker)**：WCAG/Tagged PDF 无障碍检测（免费，瑞士联邦）

> 💡 所有命令行工具均提供 macOS/Linux/Windows 安装指引，Skill 输出时会附带具体安装命令。

## ⚠️ 注意事项

1.  **备份优先**：任何优化前务必备份原始 PDF，修改不可逆
2.  **用途驱动**：印刷/屏幕/归档的优化策略互斥，务必先明确用途
3.  **版权合规**：DRM 保护或禁止编辑的 PDF 需获得授权后再处理
4.  **批量测试**：自动化脚本先在 3-5 个样本上验证，确认无误再批量执行
5.  **合规验证**：PDF/A 与 WCAG 合规需使用专用验证工具（VeraPDF/PAC），Skill 仅提供初步诊断
6.  **字体许可**：嵌入字体前确认许可证允许嵌入，避免法律风险
