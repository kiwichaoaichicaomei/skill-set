# Word 文档优化专家 (word-doc-optimizer)

## 📖 简介

这是一个专为 Aone Copilot 设计的 Skill，旨在帮助用户全方位优化 Word 文档。无论是技术文档、商务报告还是学术论文，该 Skill 都能从**结构、内容、格式、可读性**四个维度提供专业的诊断与优化建议。

## ✨ 核心功能

-   **结构诊断**：自动检测标题层级混乱、章节失衡、目录缺失等问题
-   **内容润色**：修正语病、统一语气、精简冗余、转换主动语态
-   **格式规范**：检查字体、行距、列表、表格、图表的规范性
-   **可读性提升**：长句拆分、信息列表化、关键点高亮、摘要提炼
-   **分级报告**：按 🔴高/🟡中/🟢低 优先级输出问题清单与修改对比

## 🚀 快速开始

### 1. 安装 Skill
将本目录复制到你的项目或全局 Skill 目录中：
```bash
# 项目级（仅当前项目可用）
cp -r word-doc-optimizer /your-project/.aone_copilot/skills/

# 全局级（所有项目可用）
cp -r word-doc-optimizer ~/.aone_copilot/skills/
```

### 2. 使用方式
在 Aone Copilot 对话中直接输入以下指令即可触发：

-   "帮我优化这份 Word 文档"
-   "润色一下这个技术方案"
-   "检查文档格式是否规范"
-   "让这份报告更专业易读"
-   直接上传 `.docx` / `.doc` 文件并附带优化需求

### 3. 输出示例
Skill 将返回结构化的诊断报告，包含：
-   文档类型与目标读者分析
-   整体评分（0-100）
-   分级问题清单（含位置定位）
-   具体的修改前后对比
-   可选的完整重写版本

## 📋 适用场景

| 场景 | 说明 |
|------|------|
| 技术文档评审 | 确保架构设计、API 文档逻辑清晰、术语准确 |
| 商务报告提交 | 提升报告专业性，统一品牌语调 |
| 论文/标书润色 | 符合学术/招标规范，增强说服力 |
| 内部手册更新 | 提高可读性，降低新员工学习成本 |
| 跨部门协作 | 消除歧义，确保信息传递准确 |

## ⚙️ 自定义配置

如需调整优化侧重点，可编辑 `skill.md` 中的以下部分：

-   **角色定位**：修改为特定行业专家（如"医疗文档专家"、"法律文书顾问"）
-   **核心能力**：增删特定维度的检查项
-   **输出格式**：调整报告模板以匹配团队规范
-   **触发条件**：添加更多激活关键词

## ⚠️ 注意事项

1.  **隐私安全**：上传文档前请确认已脱敏，避免泄露敏感信息
2.  **保留原意**：优化建议仅供参考，最终修改需由作者确认
3.  **格式兼容**：部分高级格式建议可能依赖特定 Word 版本
4.  **大文件处理**：超过 50 页的文档建议分章节提交优化
